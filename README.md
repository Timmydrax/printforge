# PrintForge

A web app for **3D printing enthusiasts** to browse different 3D models to print for themselves.

## Overview

This project allows users to:

- Explore a variety of 3D models
- View details about each model
- Easily find models to 3D print

It’s designed for hobbyists, makers, and anyone interested in 3D printing.

## Features

- Browse 3D models by category
- Search for models
- View model details and previews

## Tech Stack

- [Next.js](https://nextjs.org/) – React framework for building the web app
- [React](https://reactjs.org/) – Frontend library
- [pnpm](https://pnpm.io/) – Package manager
- Other libraries (to be added)
